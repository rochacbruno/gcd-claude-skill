# Cloud Logging documentation 



[Read product documentation](https://berlin.devsitetest.how/logging/docs/overview)

















**Cloud Logging** is a fully managed service that allows you to
store, search, and analyze logging data from
Google Cloud Dedicated in Germany. Logging includes storage for logs through
[log buckets](/logging/docs/buckets), a user interface called the
[Logs Explorer](/logging/docs/view/logs-viewer-interface), and
[an API](/logging/docs/reference/api-overview) to manage logs programmatically.
Logging lets you read and write log entries, query your logs, and control how you
route and use your logs.

[Go to the Cloud Logging product page for more.](https://berlin.devsitetest.how/logging/)
