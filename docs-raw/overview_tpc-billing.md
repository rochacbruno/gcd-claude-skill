## What is Cloud Billing? 

Cloud Billing is a collection of tools that help you track and
understand your Google Cloud Dedicated in Germany spending, pay your
bill, and optimize your costs. Because
Google Cloud Dedicated is a local, isolated cloud
based on Google Cloud, [not all Cloud Billing features
that are available in Google Cloud can be used in
Google Cloud Dedicated](/billing/docs/tpc-differences).

A Cloud Billing account defines who pays for a given set of
Google Cloud Dedicated resources. To use
Google Cloud Dedicated services, you must have a valid
Cloud Billing account, and must link it to your
Google Cloud Dedicated project(s). Multiple projects
can have their resource usage billed to the same account.

You can learn more about Cloud Billing in its [documentation](/billing/docs/concepts).
