# Identity and Access Management documentation 



[Read product documentation](https://berlin.devsitetest.how/iam/docs/overview)


















Identity and Access Management (IAM) lets you create and manage permissions for
Google Cloud Dedicated in Germany resources. IAM unifies access control for
Google Cloud Dedicated services into a single system and presents a consistent set of operations.




[Go to the Identity and Access Management product page for more.](https://berlin.devsitetest.how/security/products/iam)
