# Cloud SQL documentation 



[Read product documentation](https://berlin.devsitetest.how/sql/docs/introduction)


















Cloud SQL is a fully-managed database service that helps you set up, maintain, manage, and administer your relational databases on Google Cloud Platform.




You can use Cloud SQL with [MySQL](/sql/docs/mysql) or [PostgreSQL](/sql/docs/postgres).




Not sure what database option is right for you? Learn more about our [database services](https://berlin.devsitetest.how/products/databases).



[Go to the Cloud SQL product page for more.](https://berlin.devsitetest.how/sql/)
